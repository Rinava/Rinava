### About

Six years building software in regulated, high-stakes domains: healthcare billing, fintech payments, the kind of systems where a small mistake has real consequences. These days I build LLM-powered products with Claude and the Vercel AI SDK, I work end to end rather than only the frontend, and I have a habit of catching the risks others miss, like a HIPAA compliance gap I found at design time, before any code shipped.

Cat enthusiast and dog fan, building from Buenos Aires and sharing tools (often open source) along the way.

Open to remote Tech Lead and AI Product Engineer roles, US overlap.

---

### What I'm building

- **[Pagewise](https://laramateo.com):** a Claude-powered reader that summarizes any web page at three depths (brief, detailed, or ELI5), with grounded follow-up Q&A and shareable, read-only links. Free, no signup.
- **[Marksight](https://github.com/Rinava/marksight):** an open-source Markdown editor with live preview (React, Next.js, CodeMirror) that grew to 200+ users organically, through search and referrals.
- More AI experiments and side projects live on **[laramateo.com](https://laramateo.com)**.
